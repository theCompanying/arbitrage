# Arbitrage Business

Finding goods on AliExpress and selling them on Amazon.com.

## Goal

Create a profitable business by identifying winning products on AliExpress and reselling them on Amazon with proper margins.

## Structure

```
life/
├── projects/      # Active initiatives with deadlines
├── areas/         # Ongoing responsibilities
├── resources/     # Reference material
└── archives/      # Completed/inactive items

agents/            # Agent configurations and instructions
docs/              # Company documentation
memory/            # Knowledge base (PARA method)
plans/             # Strategic plans
```

## Quick Start

```bash
# Install dependencies
pnpm install

# Run tests
pnpm test

# Build
pnpm build
```

## Development

- **Project Management**: Paperclip (issue tracking and agent coordination)
- **Memory System**: PARA method (Projects, Areas, Resources, Archives)
