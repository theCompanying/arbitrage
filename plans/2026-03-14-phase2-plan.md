# Phase 2 Plan - Product Research Pipeline

**Created:** 2026-03-14  
**Owner:** CEO  
**Status:** Draft

---

## Context

MVP infrastructure complete (2026-03-14):
- Next.js/TypeScript app: `apps/web/`
- Prisma schema: `packages/database/` with Product, Supplier, Inventory, Order, ResearchNote models

## Phase 2 Goal

Build automated product research pipeline to identify profitable AliExpress→Amazon arbitrage opportunities.

## Priority Tasks

### 1. API Integrations (Week 1)

**Amazon Product Advertising API**
- Apply for API access
- Implement product lookup by ASIN
- Fetch BSR, reviews, pricing, ratings
- Cache results to avoid rate limits

**AliExpress API**
- Product search by category/keyword
- Fetch pricing, shipping, MOQ, supplier info
- Track product availability

**Keepa API (optional but recommended)**
- Historical price tracking
- BSR history
- Sales rank estimates

### 2. Margin Calculator (Week 1-2)

**FBA Fee Calculator**
- Referral fees (category-based, typically 8-15%)
- FBA fulfillment fees (weight/size tiers)
- Storage fees (monthly, seasonal)
- Inbound shipping estimates

**Profit Calculator**
- Landed cost: product + shipping + customs
- Total Amazon fees
- Net margin % and profit per unit
- Break-even analysis

**Scoring System**
- Profitability score (margin, demand, competition)
- Risk factors (review count, brand dominance)
- Recommendation engine

### 3. Product Research Dashboard (Week 2-3)

**Dashboard Features**
- Product candidates list with filters
- Sort by margin, BSR, demand score
- Quick view: AliExpress vs Amazon comparison
- One-click product import from AliExpress URL

**Product Detail View**
- Side-by-side comparison
- Fee breakdown visualization
- Historical trends (Keepa charts)
- Supplier info and contact

**Workflow Management**
- Status tracking: Research → Evaluating → Approved → Launched
- Research notes and tags
- Sample order tracking

### 4. Manual Product Research (Parallel)

While building automation:
- Identify 10-15 candidate products manually
- Use Helium 10 / Jungle Scout free trials
- Order samples from top 3-5 suppliers
- Evaluate quality, packaging, shipping time

## Success Criteria

- [ ] Can import product from AliExpress URL
- [ ] Auto-fetches Amazon data for matching ASIN
- [ ] Calculates accurate margin including all fees
- [ ] Ranks products by profitability score
- [ ] Identify 3-5 viable products in 2 weeks

## Technical Notes

**Rate Limits**
- Amazon PA-API: 1 request/second, 86,400 requests/day
- Keepa: 1 token/10 seconds (free tier)
- Implement caching layer (Redis)

**Database**
- Run migrations: `pnpm db:migrate`
- Seed script for demo data

**Deployment**
- Vercel for Next.js frontend
- Railway/Render for PostgreSQL
- Consider Docker for local dev consistency
